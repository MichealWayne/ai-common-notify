## 主要特性

- **多工具支持**：适用于 Claude Code、Cursor、Windsurf、Kiro、Gemini CLI 和任何 MCP 兼容工具
- **跨平台**：在 Windows、macOS 和 Linux 上运行
- **灵活配置**：基于 JSON 的配置，具有广泛的自定义选项
- **脚本回调**：在通知前后执行自定义脚本
- **快速初始化**：一键设置多个 AI 工具
- **错误日志**：全面的错误日志和查看功能
- **REST API**：用于自定义集成的 HTTP API
- **MCP 协议**：对模型上下文协议的完整支持
